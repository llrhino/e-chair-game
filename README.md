## ゲーム概要

電気椅子ゲームは、2人で対戦するターン制の心理戦ゲームです。  
プレイヤーはラウンドごとに「仕掛ける側」と「座る側」を交代しながら、
相手の選択を読み合い、ポイント獲得と感電リスクのバランスを競います。

ゲームエリアには複数の椅子が配置されており、
ラウンドが進むにつれて選択肢は徐々に減っていきます。

### 基本構成

- プレイ人数：2人
- 形式：ターン制・オンライン対戦
- 椅子の数：12脚

### ラウンドの役割

各ラウンドでは、プレイヤーは以下の役割に分かれます。

- 仕掛ける側
  - 電気を流す椅子を1脚選ぶ
- 座る側
  - 実際に座る椅子を1脚選ぶ

ラウンド終了後、両者の役割は必ず交代します。

### ラウンドの流れ

1. 仕掛ける側が、電気を流す椅子を1脚選択する
2. 選択を確定する
3. 座る側が、座る椅子を1脚選択する
4. 感電判定とポイント処理を行う
5. 次のラウンドへ進む

### ポイントと感電数

- 座る側が **感電しなかった場合**
  - 座った椅子の番号に応じたポイントを獲得する
  - その椅子は「獲得済みポイント」として以降のラウンドでは使用されない
- 座る側が **感電した場合**
  - 獲得済みのポイントはすべて失われる
  - 感電数を1つ獲得する

※ システム上、ポイントは「座る側」に付与される仕様になっています。  
役割名としては直感とずれる部分がありますが、ゲーム上の挙動としてはこの仕様が正となります。

### 椅子の扱い

- 各椅子には番号が割り振られている
- 座る側が感電を回避した場合、その椅子はポイントとして獲得され、以降は選択不可となる
- 感電した場合、椅子は消費されず、次のラウンドでも残り続ける
- ラウンドが進むにつれて、未使用の椅子は減少していく

### 勝利条件

以下のいずれかを満たした時点で、ゲームは即終了します。

- いずれかのプレイヤーのポイントが **40ポイント以上** に到達した場合  
  → そのプレイヤーの勝利

- いずれかのプレイヤーの感電数が **3** に達した場合  
  → そのプレイヤーの敗北  
  → 相手プレイヤーの勝利

- 最大ラウンド終了、または残りの椅子が1脚のみになった場合  
  → 両者のポイントを比較し、
  - ポイントが高いプレイヤーの勝利
  - 同点の場合は引き分け## ゲーム概要

- プレイ人数：2人
- ゲーム形式：ターン制・オンライン対戦
- 勝利条件はいずれかを満たした時点でゲーム終了
